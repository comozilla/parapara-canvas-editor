# parapara-projection-mapping

## 準備
### 必要なもの
- node.js
- bower (npm install -g bower)

### モジュールをインストール

```sh
$ npm install
$ bower install
```

### webpack でビルド

```sh
$ webpack
```

### Lint(ソースコードのチェック)
```sh
$ npm run lint
```

### webpack-dev-server を使ってビルド

```sh
$ npm run build-dev-server
```
